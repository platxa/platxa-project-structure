# Lint

Run the project linter and auto-fix where possible.

## Commands
```bash
# Check for issues
{{LINT_COMMAND}}

{{LINT_FIX_COMMAND}}
```

## After running:
- Report number of issues found and fixed

- Never suppress lint warnings without justification
